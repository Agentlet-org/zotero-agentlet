  # Agentlet for Zotero

  Run [Agentlet](https://github.com/Agentlet-org/agentlet) AI agents in Zotero 8.

  ## Installation

  1. Download the latest `.xpi` from [Releases](../../releases)
  2. In Zotero: Tools → Plugins → ⚙️ → Install Plugin From File
  3. Select the downloaded `.xpi` file
  4. Restart Zotero

  ## Requirements

  - Zotero 8+
  - An OpenAI API key (for AI-powered agents)

  ## Configuration

  1. Go to Edit → Settings → Agentlet
  2. Enter your OpenAI API key
  3. (Optional) Configure Ollama for local inference

  ## Installing Agents

  1. Click the Agentlet icon in the toolbar
  2. Click "Install Agent"
  3. Select a `.agentlet` file

  Find example agents in the [main repository](https://github.com/Agentlet-org/agentlet/tree/main/examples).
